bg-sound + GM.DLS

Congratulations for the curious...

This is where I host a custom instance of bg-sound but with a GM.DLS conversion from Rawit at VOGONS, which should almost match the Windows default sound set.

How to use


<script src="https://zoomten.github.io/gmdls/bg-sound.js"></script>
<bg-sound src="FULL URL HERE" baseurl="https://zoomten.github.io/gmdls/"></bg-sound>
Patch set samples possibly in the grey area, use at own risk.
