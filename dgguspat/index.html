bg-sound + DGGUSPAT

Congratulations for the curious...

This is where I host a custom instance of bg-sound but with the DGGUSPAT patchset, which offers a better coverage of General MIDI than Freepats.

How to use


<script src="https://zoomten.github.io/dgguspat/bg-sound.js"></script>
<bg-sound src="FULL URL HERE" baseurl="https://zoomten.github.io/dgguspat/"></bg-sound>
Patch set samples possibly in the grey area, use at own risk.
